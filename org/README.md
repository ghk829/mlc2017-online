# 클라우드 이용법

1. 클라우드 콘솔에 들어감
2. python 코드가 뭉쳐진 폴더의 한 단계 위의 폴더로 위치 이동 (local은 임의적으로 정할 수 있기 때문에 상관없음) -- 왠만하면, 클라우드로 돌리기
3. 예제에 적혀있는 파일들을 실행하면 클라우드에서 알아서 실행됨
